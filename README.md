# Commodity Price Prediction Website

This project is a web application for predicting commodity prices. It uses a React frontend and a Python (Flask) backend.

## Getting Started

### Prerequisites

- Python 3.x
- Node.js and npm

### Backend Setup

1. Navigate to the `backend` directory:

   ```sh
   cd backend
   ```

2. Install the required Python packages:

   ```sh
   pip install -r requirements.txt
   ```

3. Run the Flask application:
   ```sh
   python app.py
   ```

### Frontend Setup

1. Navigate to the `frontend` directory:

   ```sh
   cd frontend
   ```

2. Install the required npm packages:

   ```sh
   npm install
   ```

3. Run the React application:
   ```sh
   npm run dev
   ```

## Usage

1. Open your browser and navigate to `http://localhost:5173` (or the port specified in the terminal).
2. Select a commodity from the dropdown menu.
3. Choose the number of months you want to predict.
4. Click the "Predict" button to see the forecast.
